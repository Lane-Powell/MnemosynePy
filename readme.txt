Standard field abbreviations:

a = Attribution (author)
t = Title
r = rating
n = Edition Notes
c = Comments

Commands:

search [field abbreviation] [search term]
- Searches selected field in all texts in current library for the search term. Returns a list of texts in the display case.

open [display case index]
- Displays data from a given record retreived by a search.

edit [display case index] [field]
- Displays prompt to edit field in given record.

new
- Displays prompt to create a new record.

commit
- Saves new/modifed data (done automatically when the user closes Mnemosyne with the quit or exit command).

help
- Reminds user that the readme exists.

quit (or exit)
- CLoses Mnemosyne and saves new/modified data.
